== About me
I'm Alberto Irurueta Carro, a software engineer with a soft spot for turning equations into working
code. Most of my time goes into building the open-source Java libraries documented on this very site --
covering everything from linear algebra, statistics, sorting and numerical methods, to more applied work
in computer vision, 3D reconstruction and augmented reality.

When I'm not chasing eigenvalues or wrangling point clouds, you'll find me tinkering with things like
indoor Wi-Fi based navigation or an AR app that estimates shoe size from a photo -- proof that geometry
is good for more than homework. I care about code that is well tested, well documented and genuinely
useful, which is, not entirely by coincidence, why this documentation site exists in the first place.

You can find more about me, and all these projects, on
https://github.com/albertoirurueta/albertoirurueta[Github].
