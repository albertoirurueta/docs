Metric and imperial helpers

Several unit enums expose helpers for classifying units:

import com.irurueta.units.DistanceUnit;
import com.irurueta.units.UnitSystem;

UnitSystem system = DistanceUnit.getUnitSystem(DistanceUnit.MILE);
boolean metric = DistanceUnit.isMetric(DistanceUnit.KILOMETER);
boolean imperial = DistanceUnit.isImperial(DistanceUnit.FOOT);

DistanceUnit[] metricDistanceUnits = DistanceUnit.getMetricUnits();
DistanceUnit[] imperialDistanceUnits = DistanceUnit.getImperialUnits();

These helpers are available on families where the distinction applies, such as distance, speed, surface, volume, weight, acceleration, and temperature.
