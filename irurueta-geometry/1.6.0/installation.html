Installation

This documentation was generated with the assistance of AI. Please report any inaccuracies.

irurueta-geometry is published to Maven Central. Add it as a dependency to your Maven pom.xml.

Latest release

<dependency>
    <groupId>com.irurueta</groupId>
    <artifactId>irurueta-geometry</artifactId>
    <version>1.6.0</version>
    <scope>compile</scope>
</dependency>

Latest snapshot

<dependency>
    <groupId>com.irurueta</groupId>
    <artifactId>irurueta-geometry</artifactId>
    <version>1.7.0-SNAPSHOT</version>
    <scope>compile</scope>
</dependency>

Snapshot versions are published to the Sonatype snapshots repository; make sure your project’s repositories include it if you depend on a -SNAPSHOT version.

Requirements

  • Java 21 or later.
